# Sticker Pack Game

A collectible sticker game built with HTML, CSS, and JavaScript.

## Features
- User authentication with Supabase
- Collect 700 unique stickers
- Open sticker packs (5 cards per pack)
- Track collection and duplicates
- Sell duplicates for coins
- Coin flip mini-game
- Cloud save functionality
- Responsive design for mobile and desktop
